## 리턴값으로 초기화하기

```
int sum = addTwoNumbers(1, 3);
```

이런 식으로 함수의 리턴값으로 초기화할 수도 있음

그렇다고 함수 안에서 함수를 선언할 수는 없음
클래스 안에서 클래스 선언하는 건 가능하다.
